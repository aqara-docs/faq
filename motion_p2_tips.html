22  모션/조도 센서 P2

22.1 설치

질문 1: 모션/조도 센서 P2의 설치 방법을 알 수 있을까요?

답변 1:

모션/조도 센서 P2의 설치 영상과 매뉴얼은 아래 링크를 클릭하시면 보실 수 있습니다.

모션/조도 센서 P2

22.2 사용 팁

질문 1: 모션/조도 센서 P2는 어떤 무선 프로토콜을 지원하나요?

답변 1:

모션/조도 센서 P2는 Thread와 BLE를 지원합니다.

질문 2: 모션/조도 센서 P2는 어떤 배터리를 사용하나요?

답변 2:

모션/조도 센서 P2는 CR123A 배터리를 사용합니다.

질문 3: 모션/조도 센서 P2를 초기화(리셋/재설정)하는 방법을 알려 주세요.

답변 3:

리셋버튼을 5초가 누르고 있다가 놓으면 페어링 모드로 들어갑니다.

질문 4: 공장 초기화하는 방법을 알려 주세요.

답변 4:

리셋 버튼을 10회 빠르게 누르면 공장 초기화 모드로 복구되고 지시등이 점멸합니다.

22.3 Troubleshoot

질문 1: 전원을 켰을 때 LED가 몇 분동안 (10분) 지시등이 계속 깜빡거리고 있습니다. 왜 그런가요?

답변 1:

모션/조도 센서 P2가 매터허브에 연결되어 있지 않기 때문입니다.

질문 2: 모션/조도 센서 P2가 절전모드에 들어 간 것 같습니다. 깨우는 (wake-up) 방법이 있나요?

답변 2:

지시등이 점멸하기 시작할 때까지 리셋 버튼을 5초간 누르고 있다가 놓으면 장치가 wake-up 및 네트워크 초기화 상태가 됩니다.
