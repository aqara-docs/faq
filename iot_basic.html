1.2 FAQ

질문 1: 공장 초기화와 네트워크 초기화의 차이는 무엇인지요?

답변 1:

공장 초기화는 모든 설정을 초기화 하는 것으로 이전 설정이 있다면 모두 삭제 되므로 초기화 시 주의하여 진행하셔야 합니다. 각 아카라 제품의 공장 초기화 방법은 다음 사이트를 참고 하세요.

공장 초기화

반면에 네트워크 초기화의 경우는 이전 설정 값들을 그대로 보존한 채 와이파이 AP에 새로 연결할 준비가 되도록 합니다. 네트워크 초기화하는 방법은 다음과 같습니다.

허브 - 10초 이상 리셋 버튼 클릭

서브 장치 - 3-5초 이상 리셋 버튼 클릭

질문 2: 공유기를 교체했는데, 기존에 설치된 아카라 제품들은 새로 설치해야 하는지요?

답변 2:

공유기가 교체 되었다면, 아카라 허브 및 와이파이 제품들은 새로 설치하셔야 합니다. 단, 다음과 같은 방법을 이용하면 모두 설치하실 필요는 없습니다.

  • 새공유기의 AP 이름과 패스워드를 기존 공유기와 동일하게 설정: 아카라 제품을 새로 설치하실 필요 없습니다.

  • 아카라 허브를 네트워크 초기화 후 설치: 아카라 허브에 연결되어 있는 아카라 하부 장치들은 새로 연결하실 필요 없습니다.

  • (주의) 만약, 아카라 허브를 공장 초기화 하셨다면 모두 새로 설치 하셔야 합니다.

질문 3: 아카라 제품들을 구매했습니다. 사용하기 위해서는 무엇을 해야 하는지요?

답변 3:

아카라 허브를 아카라 홈앱에 설치하는 작업부터 하셔야 합니다. 기본 설정 순서는 다음과 같습니다.

  • 허브 바인딩 :허브를 인터넷에연결하는 작업

  • 하위 기기 페어링: 센서, 커튼컨트롤로와 같은 동작 기기(Actuator)를 허브에 연결하는 작업

  • 동작기기 기본 설정 : 커튼, 블라인드와 같은 기본설정이 필요한 기기의 기본설정, Ex)블라인드 상하한선 설정, 커튼 제한선 설정

  • 자동화(실행)설정 : 센서나 버튼과 같은 입력기기의 조건이 조건에 맞으면 지정한 기기(Actuator)가 동작하도록 하는 기능

  • 동시 실행 설정 : 등록한 기기의 동작들이 앱에서 클릭하거나 음성으로 명령할 때 일관 실행하도록 하는 기능

  • 바로가기 추가 : 홈 화면에 바로가기 탭에 자주 사용하는 제어기기나 센서를 한곳에 모아 관리하도록 하는 편의 기능으로 필요시 설정해 사용

  • 보안모드 설정 : 집을 비웠을 때 소리, 움직임 등이 감지되면 경보음 재생 및 메시지 알림을 하는 기능으로 필요 시 설정해 사용

질문 4: 허브 바인딩하는 과정에서 문제가 발생했습니다. 무엇을 체크해야 하는지요?

답변 4:

  • 허브 전면의 버튼을 빠르게 10회 눌러 완전 초기화

  • 연결하려는 신호 :2.4GHz신호 맞는지 재확인

  • WiFi아이디와 비밀번호가 정확한지 재확인

    (※영문 또는 숫자로만 이루어진 것이 좋음 /일부 특수문자는 연결 방해)

  • 공유기 보안 설정 WAP2여부 확인

    (※WPA/WPA2혼합 orWPA3는 지원 X)

  • 공유기 설정 중 와이파이 무단 사용 방지 또는 AP분리 기능 등 연결 방해 특수한 보안 설정 해제

  • 앱의 지시대로 처음부터 장치 연결 시도

[추가 점검 사항 - 연결 실패시]

  • 허브 연결 대기중인 상태 (노란불 깜빡임)에서 휴대폰 와이파이 ON

    lumi-gateway xxxx 와이파이 신호가 정상적으로 보이고 연결 가능 여부 확인

  • 허브 연결된 전원 공급 장치 5V 1A (E1 허브 5V 0.5A) 규격에 맞는지 확인

    (※가능하다면, 전원 공급 장치 변경 시도)

[로컬 네트워크의 문제 외에 애플 홈킷 오류로 인해 연결이 불가한 경우]

※제품에 붙어있는 홈킷 QR코드 촬영을 시작으로 진행되는 절차 -모두 애플 홈킷을 통한 연결과정

  • 오류 메시지 :악세서리를 추가할 수 없음 /악세서리가 이미 추가 되어 있음 /악세서리 검색 실패 등

    ⇒ iOS장치 (휴대폰 또는 태블릿)재부팅 후 허브 전면 버튼 빠르게 10회 눌러 초기화

    3분 정도의 대기 시간 후 다시 시도

  • 참고 사항

    ① M2허브 –추가 화면 우측 상단 연결 방식을 블루투스로 변경 시,

    홈킷에 등록하지 않고 아카라 홈 앱에만 장치 추가하는 방식으로 우회 가능 

    ② 홈킷 연결 필요하지 않은 환경 –추가 화면 우측 안드로이드 휴대기기로 연결 후 애플기기에서도 동일하게 사용 가능

[문제 지속되는 경우]

  • 휴대폰 핫스팟 켜서 해당 신호에 허브 연결 시켜보기 (장치 연결 기능 자체 문제 여부 점검)

  • 방법 : 핫스팟 연결 가능 시, 공유기를 잠시 꺼두고 핫스팟 신호의 아이디와 비밀번호를 사용하시려는 공유기의 신호와 동일하게 설정 

  • 연결 완료 후, 핫스팟 삭제하여 장치가 공유기 쪽으로 자연스럽게 연결되도록 유도 

질문 5: 새로운 허브를 추가했는데 기존에 설치된 허브에 연결된 몇몇 아카라 디바이스의 연결이 끊겼습니다. 원인이 무엇일까요?

답변 5:

기존 허브의 Zigbee 채널과 새로 추가된 허브의 채널을 비교해 보시기 바랍니다.

허브 메인 화면 >> 네트웍 정보 >> Zigbee 채널에서 확인하실 수 있습니다. 만약 Zigbee 채널이 같다면 새로운 허브를 공장 초기화 하신 후 다시 연결해 보십시오. 채널이 달라야 정상 동작할 수 있습니다. 만약 채널이 다른데도 불구하고 이러한 증상이 발생한다면, 2.4Ghz 대역의 다른 신호(와이파이, 블루투스등) 새로운 허브 추가 대신 Zigbee 리피터(스마트 플러그, (중성선이 연결된) 상시 전원 디바이스)를 추가해 보시기 바랍니다.

질문 6: 허브를 공장 초기화 한 후 아카라 디바이스들을 연결하려고 하는데 일부는 연결되었는데, 일부는 전혀 연결이 안되고 있습니다. 어떻게 해야 하는지요?

답변 6:

  • 댁내 모든 허브와 연결이 안되는 아카라 디바이스의 전원 (또는 배터리) 차단, 1분 이상 충분히 대기

  • 연결이 안되는 아카라 디바이스의 전원 (또는 배터리) 연결 후 리셋버튼 5초이상 눌러 페어링 모드로 잘 진입하는지 확인
